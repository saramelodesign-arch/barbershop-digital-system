# Product Vision

## Problem

Barbearias pequenas gerem marcações via:
- WhatsApp
- Chamadas
- Agenda em papel

Isto gera:
- Conflitos de horários
- Perda de marcações
- Falta de controlo sobre disponibilidade
- Dependência constante do telefone

## Target User (Fase 1)

- Barbearia local independente
- 1 a 5 barbeiros
- Sem software de gestão
- Gestão manual ou semi-manual

## Target User (Fase 2 – SaaS)

- Barbearias pequenas e médias
- Múltiplas localizações
- Necessidade de controlo de agenda e serviços

## Value Proposition

Sistema simples de gestão de marcações que:
- Evita conflitos
- Organiza agenda por barbeiro
- Permite configuração de horários e serviços
- Pode escalar para múltiplas barbearias

## Version 1 Goal (Pilot)

Validar que:
- Marcações funcionam sem conflitos
- Configuração é simples
- Agenda é clara e utilizável

## Success Metrics (MVP)

- Criar negócio
- Criar barbeiros
- Criar serviços
- Criar marcações sem conflitos
- Cancelar respeitando regras
- Usado durante 2 semanas reais

## Non-Goals (fora do MVP)

- Pagamentos online
- Notificações automáticas
- App mobile
- Marketplace
- Gestão financeira avançada
