# Roadmap

## Phase 0 – Setup

- Criar repositório
- Definir stack
- Configurar projeto
- Definir schema base

---

## Phase 1 – Core Structure

- Multi-tenant base
- Modelos:
  - Barbershop
  - Barber
  - Service
  - Booking
- Seed inicial

Critério de sucesso:
DB estruturada e consistente.

---

## Phase 2 – CRUD Management

- CRUD Services
- CRUD Barbers
- Business configuration screen

Critério de sucesso:
Admin consegue configurar tudo.

---

## Phase 3 – Booking Engine (parte crítica)

- Criar marcação
- Validar conflitos
- Validar horários
- Cancelar marcação

Critério de sucesso:
Impossível criar sobreposição.

## Phase 4 – Calendar View

- Visualização por barbeiro
- Visualização por dia
- Estados visuais

## Phase 5 – Pilot Test

- Instalar na barbearia real
- Usar durante 2 semanas
- Recolher feedback

## Phase 6 – Preparação SaaS

- Subdomínio por barbearia
